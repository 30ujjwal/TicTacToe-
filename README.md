# Tic-Tac-Toe Game (Java)

A classic 2-player Tic-Tac-Toe game implemented in Java. You can play this game with your friend on the same computer through the terminal/console.

## Features
* **2-Player Mode:** Play with your friend turn by turn.
* **Simple Interface:** Clean and easy-to-understand console layout.
* **Win/Draw Detection:** Automatically detects when a player wins or if the game is a draw.

## How to Run
1. Download or clone this repository.
2. Open the project in your favorite IDE (like IntelliJ IDEA or Eclipse) or terminal.
3. Run the `Main.java` file.
